# 📊 Mocktest Economics Web Application (Gamified Edition)

Mocktest วิชาเศรษฐศาสตร์ออนไลน์ ที่ออกแบบมาเพื่อให้การเรียนน่าสนุก และมีประสิทธิภาพมากขึ้น! 
ถูกพัฒนาขึ้นโดยใช้แนวคิด **Gamification** และ **จิตวิทยาการเรียนรู้ (Active Recall & Spaced Repetition)** 
เพื่อกระตุ้นให้ผู้ใช้งานอยากกลับมาทบทวนเนื้อหาอย่างต่อเนื่อง แม้จะมีเวลาน้อยแต่ก็ยังสามารถเรียนได้
**อย่าปล่อยให้เวลาเล็กๆ น้อยต้องสูญเปล่า**

โปรเจกต์นี้เขียนด้วย **Vanilla JavaScript** ล้วนๆ ไม่พึ่งพา Framework หนักๆ ทำงานได้รวดเร็ว และสามารถรันบนเบราว์เซอร์ได้ทันทีโดยไม่ต้องตั้งค่า Server

---

## ✨ Features (ความสามารถเด่น)

### 🧠 ระบบทดสอบอัจฉริยะ (Smart Quiz System)
- **Dynamic Question Selection:** สามารถเลือกจำนวนข้อสอบ และเน้น "สุ่มบทที่ต้องการทำเป็นพิเศษ" ได้ (ระบบถ่วงน้ำหนักข้อสอบ)
- **Time Simulation:** จำลองสภาพแวดล้อมการสอบจริง ด้วยระบบคำนวณเวลาและจับเวลาถอยหลังแบบ Real-time
- **Hint System:** ระบบคำใบ้ช่วยเหลือ พร้อมจำกัดจำนวนครั้งการใช้งานเพื่อความท้าทาย

### 📊 ระบบวิเคราะห์ผลลัพธ์ (Performance Analytics)
- **Ghost Line Radar Chart:** แสดงกราฟใยแมงมุม (Radar Chart) วิเคราะห์ความแม่นยำรายบท พร้อม **"เส้นกราฟเปรียบเทียบกับรอบที่แล้ว"** เพื่อให้เห็นพัฒนาการอย่างชัดเจน
- **Weakness Detection:** ระบบวิเคราะห์และแจ้งเตือน "บทที่เป็นจุดอ่อน (ควรทบทวนด่วน)" และ "บทที่เป็นจุดแข็ง" ทันทีหลังสอบเสร็จ

### 🎮 Gamification & การกระตุ้นแรงจูงใจ
- **🔥 Study Streak System:** ระบบนับวันทบทวนต่อเนื่อง (Local Storage) ยิ่งทำต่อเนื่อง ป้ายไฟจะยิ่งอัปเลเวลและเปลี่ยนสีได้ (ส้ม ➔ ม่วง ➔ ฟ้า ➔ เขียวมรกต)
- **🎉 Confetti Celebration:** แอนิเมชันจุดพลุกระดาษฉลองความสำเร็จ เมื่อทำคะแนนได้เกิน 80% หรือทำคะแนนได้ดีกว่ารอบที่แล้ว
- **💬 Dynamic Feedback:** ระบบสุ่มคำชมและคำแนะนำกว่า 50 รูปแบบ ที่จะเปลี่ยนไปตามช่วงเปอร์เซ็นต์คะแนนที่ทำได้
- **📢 Share Score:** ปุ่มกดคัดลอกผลคะแนนและจุดแข็งแบบอัตโนมัติ พร้อมนำไปอวดเพื่อนในโซเชียลมีเดียได้ทันที

### 🃏 ระบบทบทวนเพื่อความจำระยะยาว (Active Recall)
- **3D Anki-Style Flashcards:** เปลี่ยนเฉลยข้อสอบที่น่าเบื่อให้เป็น "การ์ด 3 มิติ พลิกหน้า-หลังได้" เพื่อการ (Active Recall) 
ก่อนนำเมาส์ไปชี้เพื่อดูเฉลยและคำอธิบายด้านหลัง

---

## 🛠️ Tech Stack

- **Frontend Structure:** HTML5
- **Styling & UI:** Tailwind CSS (ผ่าน CDN) / Glassmorphism Design
- **Logic & State Management:** Vanilla JavaScript (ES6+)
- **Data Visualization:** Chart.js (ผ่าน CDN)
- **Animations:** Canvas-Confetti (ผ่าน CDN) / CSS Custom Keyframes
- **Storage:** Browser `localStorage` และ `sessionStorage` (บันทึกข้อมูลและสถิติข้ามวันได้โดยไม่ต้องมี Database)
